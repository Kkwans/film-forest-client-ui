# SPEC -- 影视森林全面打磨二期

## 1. 目标与价值

- **解决什么问题**: 影视森林项目存在大量 UI/UX 问题和数据质量问题，影响用户体验
- **为什么要做**: 这是一个面向用户的影视资源聚合网站，体验差会直接流失用户
- **预期收益**: 全面提升用户体验，修复已知问题，发现并解决未知问题

## 2. 上下文与约束

- **技术栈**: SpringBoot3 + MyBatis-Plus3 + MySQL8 + Next.js
- **架构**: 用户端(client-ui/client-server) + 管理端(admin-ui/admin-server)
- **部署**: NAS Docker 容器（ARM64）
- **约束**: 遵循 SDD 流程，每 30 分钟执行一轮

## 3. 功能需求

### 用户端问题（client-ui/client-server）

#### F1: 在线观看列表区分平台来源
- **描述**: 当前所有平台的播放链接混在一个列表里，无法区分来源
- **输入**: 用户进入详情页
- **输出**: 按平台分组展示，每个平台有独立的标识
- **优先级**: P0

#### F2: 评价详情弹窗样式优化
- **描述**: 查看评价详情不应复用新增评价的输入框样式，应为只读展示
- **优先级**: P0

#### F3: 评价编辑功能修复
- **描述**: 点击编辑后数据被清空，应保留原有数据
- **优先级**: P0

#### F4: 详情页布局优化
- **描述**: 左侧海报高度应与右侧信息区一致，避免信息区出现在海报下方
- **优先级**: P1

#### F5: 爬虫数据质量问题
- **描述**: 简介数据不完整（有些包含"[展开全部]"按钮文本）、在线观看列表未分类
- **优先级**: P0

### 管理端问题（admin-ui/admin-server）

#### F6: 爬虫配置创建后不回显
- **描述**: 新建配置后页面无反馈，刷新后列表为空
- **优先级**: P0

#### F7: 爬虫配置表单优化
- **描述**: 类型筛选选项混乱（语言混入类型筛选）、定时规则跳转 bug
- **优先级**: P0

#### F8: 内容管理筛选项被遮挡
- **描述**: 全部分类/全部状态下拉列表被遮挡
- **优先级**: P1

#### F9: 内容表单字段不完整
- **描述**: 新增/编辑内容缺少海报配置
- **优先级**: P1

#### F10: 数据统计页面优化
- **描述**: 柱状图样式问题、Tooltip 从左上角飞出、可扩展图表类型
- **优先级**: P1

#### F11: 资源管理界面优化
- **描述**: 磁力/网盘列表展示不全、无筛选/编辑/分页、按钮文字颜色问题
- **优先级**: P0

#### F12: 仪表盘"下线"按钮含义不清
- **优先级**: P2

#### F13: 自主发现的问题
- **描述**: 梳理代码自行发现更多问题
- **优先级**: P1

## 4. 非功能需求

| 维度 | 要求 |
|------|------|
| 质量 | 每个修复必须验证，不能引入新问题 |
| 测试 | 修改后必须测试，发现问题及时修复 |
| 完整性 | 不仅修复列出的问题，还要自主发现更多 |

## 5. 验收标准

| 编号 | 场景 | 预期结果 |
|------|------|----------|
| AC1 | 在线观看列表 | 按平台分组展示 |
| AC2 | 评价详情弹窗 | 只读展示，无输入框 |
| AC3 | 评价编辑 | 保留原有数据 |
| AC4 | 详情页布局 | 海报与信息区等高 |
| AC5 | 爬虫配置创建 | 创建后立即回显 |
| AC6 | 类型筛选 | 无语言混入 |
| AC7 | 定时规则 | 不跳转到其他 tab |
| AC8 | 资源管理 | 有筛选/编辑/分页 |
| AC9 | 数据统计 | Tooltip 即时显示 |
